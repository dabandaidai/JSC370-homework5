## First plot

data <- read.csv("acs2017_census_tract_data.csv")
sort_by_pop <- data %>% 
  arrange(desc(TotalPop))
sort_by_pop <- head(sort_by_pop, 100)
plt1 <- plot_ly(
  sort_by_pop,
  x = ~ Income,
  y = ~ log(Unemployment),
  color = ~ State,
  size = ~ TotalPop,
  mode = 'markers',
  type = "scatter",
  sizes = c(10, 50),
  marker = list(sizemode = "area", 
                opacity = .8),
  hoverinfo = "text", 
  text = ~ paste0(
                  State, "\n",
                  "   County: ", County, "\n",
                  "   Poverty: ", Poverty, "\n",
                  "   Men: ", Men, "\n",
                  "   Women: ", Women, "\n",
                  "   MeanCommute: ", MeanCommute
                )
)

The plot shows the relationship between the income and the log unemployment rate(since the histogram of unemployment shows heavy skewness) for the top 100 most populated states in the data frame, where each point contains information about the state, mean commute time, poverty rate, number of men and number of women in the tract. The size of each point is determined by the total population of the state, and the color of the point represents the state. This plot illustrate some interesting features: * There are two tracts with particularly low unemployment rate: one in salt lake county in Utah and one in ada county in Idaho. Reasonably, these two tracts all have relatively low poverty rate and shorter mean commute time(in minutes). * As the income increases, the unemployment rate seems to be a slight tendency of decreasing, especially when income is in the range from 40k to 100k. * There does not seem to be a relationship between the population of the tract and the unemployment rate according to the size of the balls. * It seems like Texas and Florida contains the highest number of the most populated tracts.

plt2 <- plt_smooth <- ggplot(
  sort_by_pop,
  aes(x = MeanCommute, y = log(Unemployment))
) +
  geom_point(aes(colour = State, size = TotalPop)) + 
  geom_smooth() +
  theme_minimal() +
  scale_x_continuous(trans = "log")
ggplotly(plt_smooth)

The plot shows the relationship between the mean commute time (in minutes) and the log unemployment rate(again, due to skewness) for the top 100 most populated states in the data frame. The size of each point is determined by the total population of the state, and the color of the point represents the state.

According to the fitted smoothed line, there are no monotone relationship between the mean commute time and unemployment rate: in fact as the mean commute time decreases from 14.7 minutes to 30 minutes, the unemployments decreases; however, as the mean commute time increases from 30 minutes to 50 minutes the unemployment rises again.


Third plot

data$abb <- state.abb[match(data$State,state.name)]
data$Unemployment2 <- data$Unemployment * 2
map <- plot_geo(
  data,
  locationmode = "USA-states"
) %>% 
  add_trace(
    z = ~ Unemployment,
    locations = ~ abb
  ) %>% 
  layout(
    geo = list(
      scope = "usa",
      showlakes = TRUE,
      lakecolor = toRGB("darkgrey")
    )
  )

From the map I observe that the unemployment rate in each state is approximately even, ranging from 0 to 20 percent. In general, the states in the middle area have lower unemployment rate than the states in the south and south east, and South Dakota seems to standout by having an unemployment rate that is obviously higher than any other states.
